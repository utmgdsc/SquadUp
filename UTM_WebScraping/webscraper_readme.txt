Libraries and Versions:
- Selenium version: 4.15.2 
- BeautifulSoup4 version: 4.12.2
- Firebase-admin: 6.3.0

Limitations:
This web scraper, like any other generic web scraper, is built using a pattern seen on the loxi.io website. Any changes to how loxi.io displays data can lead to changes in the behavior of the web scraper.

Other Notes:
    - A service key for a Firebase database is required and, for security reasons, omitted from the repository.
